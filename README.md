COVID-19 Data Analysis (Self-Contained)

This project is a **COVID-19 data analysis and visualization tool** implemented in Python.  
It uses a **synthetic dataset** generated within the script, so no external files are required.

Features

- Generate synthetic COVID-19 data for multiple countries
- Explore trends:
  - Daily new cases
  - Daily deaths
  - Vaccination progress
- Compare countries (example: India vs USA)
- Interactive Plotly visualization
- Saves all plots to `images/` folder
- Prints summary statistics

Installation

1. Clone the repository:

bash
git clone https://github.com/<your-username>/covid19-data-analysis.git
cd covid19-data-analysis

2. Install required libraries:

pip install -r requirements.txt
